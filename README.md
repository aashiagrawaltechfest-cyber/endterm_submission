# endterm_submission
# Predicting Stock Volatility Using Financial News Sentiment

## Description
This project focuses on predicting stock market volatility by analyzing the sentiment of financial news.
The notebook processes stock price data along with related news articles, extracts sentiment information,
and studies its relationship with market volatility. The goal is to understand how news sentiment can
influence stock price movements.

## Repository Contents
- Aashi_endterm.ipynb – Jupyter Notebook containing all code, data processing, sentiment analysis, and results
- README.md – Documentation describing the project and repository structure

## How to Run
1. Open the notebook using Jupyter Notebook or Google Colab
2. Run all cells sequentially from top to bottom to reproduce the analysis

## Requirements
- Python 3
- Required Python libraries (such as pandas, numpy, yfinance, and sentiment analysis libraries)
